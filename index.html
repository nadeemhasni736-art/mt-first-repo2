Explanation of HTML Tags

<html> : Defines the beginning and end of an HTML document.

<head> : Contains information about the webpage such as the title.

<title> : Sets the title shown on the browser tab.

<h1> : Displays the main heading of the webpage.

<p> : Used to write paragraphs or text content.

<img> : Displays an image on the webpage.

<a> : Creates a hyperlink to another website.

<hr> : Adds a horizontal line between sections.
